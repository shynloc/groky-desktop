市面上主流的 AI 编程助手都有哪些？帮我对比一下各自的优劣势
Groky

目前主流的 AI 编程助手可以分为 IDE 插件 和 CLI Agent 两类，各有侧重：

Grok Build CLI Agent
✓ 1M context，multi-agent，MCP
− 无原生 GUI（Groky 解决这个）
Claude Code CLI Agent
✓ 强推理，文件系统操作稳定
− 速度偏慢，token 成本高
GitHub Copilot IDE Plugin
✓ 补全体验最成熟，生态广
− Agent 能力弱，上下文窗口小
Cursor IDE
✓ UI 体验最佳，Composer 设计优秀
− 绑定 VS Code fork，无 1M ctx

偏向 CLI + 超大 context + agent 编排的话，Grok Build 目前最激进。需要深入对比某个方向？
