# Groky 代码审计文档

本目录包含 Groky 项目的安全审计报告、待办事项跟踪和代码改动记录。

## 文档结构

| 文档 | 说明 | 维护者 |
|------|------|--------|
| [AUDIT_REPORT.md](./AUDIT_REPORT.md) | 完整审计报告 | AI Agent |
| [TODO.md](./TODO.md) | 待办事项跟踪 | AI Agent |
| [CHANGELOG.md](./CHANGELOG.md) | 代码改动记录 | AI Agent |

## 使用说明

### 待办事项状态标记

- `[ ]` - 待处理
- `[/]` - 进行中
- `[x]` - 已完成
- `[-]` - 已取消
- `[!]` - 阻塞中

### 优先级标记

- `P0` - 立即修复（安全漏洞）
- `P1` - 尽快修复（重要问题）
- `P2` - 计划修复（改进项）
- `P3` - 优化项（Nice to have）

## 责任说明

- 每项改动必须记录在 CHANGELOG.md
- 完成待办事项后必须标记为 `[x]` 并注明完成日期
- 推送代码前确保文档同步更新
